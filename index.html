Server News

November 1st, 2017

Eyy, look at that, we got a new website. Welcome everyone, This is our new website and I can finally edit it freely. If you guys didn't know, I highly disliked enjin and it took quite a lot of money from the donations of the server. Of course, all of the money goes straight back into the server, but its also nice to be able to put all of the donations you guys send straight towards the server rather than having a large chunk taken towards a service that doesn't even give me options. This however also means that theres a new forums, I'll be adding a lot of posts there and various sections that'll sorta center around the new server system. I'll also add a sort of archive there that I'll add a lot of the old news posts and stuff there just incase anyone ever wanted to sorta look at our progress or whatever, can't hurt to have it.

November 1st, 2017

Big news We have decided that we're going to have a pretty big update, we're not entirely sure when it will be, but planning and some work has already started on it. I've gotta level with you guys, my absense on the server hasn't been entirely just cause of my being busy with language courses. Its primarily been because I've been kinda bored with the server. Much like most of you, I find the server fun to play and work on, but I feel like theres a sort of goal missing. I want to play, but I don't have a reason to. For a long time now, I've had a server idea I wanted to incorperate into our server, or simply make a new server with this idea. This is: Real life nation systems. I've wanted to make a server that works like real life for a long time now, a server that has its own systems of government, various towns to join, religions, trade systems, cults, assassinations, literally anything you can put your mind to, could happen. People could have their own jobs or freelance for cities. A king of one nation could hire you to assassinate the king of another. Of course, with this idea comes a lot of new plugins, many of which have been already planned out in full but will require some testing, so the first few weeks of the server opening to this new system will be in a sorta of Beta mode where things are changing often. The server will be generally a bit more challenging and not as easy as it currently is to make things more realistic. Custom races and classes will get a reworking entirely in order to be more PvP friendly. Speaking of which, there will be PvP outside of towns. The server is going to get some very big changes and I invite all of you to ask as many questions here or on discord as you like and I will answer them all when I get the chance. Join our discord, I'll be giving more details about the server there.
