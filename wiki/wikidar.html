    Dart's Tweaks

    These are a series of lot of smaller, sort of "Quality of life" style plugins. They do a range of things such as opening double doors when one is right clicked, help plant crops automatically when one is broken, and generally do little things, many of them would be things you'd find done in modpacks. Some also add things that ease the life of the staff, such as our /ticket plugin. None of these are currently working, however, they are all planned and most are created to some extent.

    Plugin name Description Commands Completed?
    Better Planting This plugin makes it so that when any crop is broken, it replants itself automatically as long as the player as the corresponding seed in their inventory. - No
    Drop Fixes This plugin makes it so that blocks that normally break and drop nothing or drop and break into pieces, now drop their item (Bookshelves, enderchests, glass, cakes, etc.) - Partial
    Combat Tweaks This adds: Blood effects when attacked and an additional boost of resistance when attacked while holding a shield. - No
    Tickets This allows players to use a command to report issues in-game. We prefer usage of Discord, but this is a fine alternative for players without Discord. /ticket help Yes
    Compression This plugin will allow players to convert all of the singular items in their inventory into the more condensed block forms automatically for easier storage. /Compress No
    Door Fix Simple: This plugin makes it so that when you right click a door to open it, if theres a door next to it, it'll open that one as well, making double doors work in one click. - No
    Noisy Heads This makes it so that decorative heads will be somewhat interactable when placed, giving off the sound of the mob it belongs to when right-clicked - No
    Armorstand-able This allows players to use a single command while looking at an armor stand to convert it into an armed armor stand, allowing players to place items in the hands. /armorstand No
    Useful Hoes This plugin will add multiple uses to hoes, such as allowing them to plant crops much quicker and allowing them to break large amounts of leaves from trees and convert them into sticks. - No
